АВТОПОЛУЧЕНИЕ ОТЧЁТА ИЗ WIALON

1. Установить зависимости:
   pip install -r requirements.txt

2. Скопировать .env.example в .env

3. В .env указать:
   WIALON_TOKEN
   WIALON_REPORT_RESOURCE_ID
   WIALON_REPORT_TEMPLATE_ID
   WIALON_REPORT_OBJECT_ID

4. Где взять ID:
   - запустить run_web_dashboard.bat
   - открыть боковую панель "Wialon автоотчёт"
   - указать токен или заполнить .env
   - нажать "Найти шаблоны и группы"
   - выбрать нужный шаблон и группу
   - при успешной проверке перенести ID в .env для автоматического запуска

5. Ручная синхронизация:
   run_wialon_sync_once.bat

6. Автоматизация на Windows:
   Планировщик заданий Windows → Создать задачу → Триггер ежедневно → Действие:
   Program/script: путь к run_wialon_sync_once.bat

Файл БД:
   data/driving_dashboard.sqlite

Скачанные XLSX-отчёты:
   data/wialon_reports/

Защита от дублей уже включена: повторный импорт пересекающегося периода не задваивает события.
